游戏王规则详解

卡片种类：

怪兽卡
基本介绍：
　　场上有相同等级的怪兽2只以上存在时,通过把那些相同等级的怪兽作为素材,进行エクシ`ズ召唤(XYZ召唤)来特殊召唤。
　　(XYZ怪兽)
　　额外卡组的黑色怪兽卡。
　　(阶级)
　　XYZ怪兽卡上相当于レベル(等级)的黑星，和等级相反是从左到右的排法。XYZ怪兽没有等级，和针对等级的卡无关。
　　例：「等级限制B地区」让4星以上的怪兽变成守备表示的效果对4阶的「希望皇 霍普」没有影响。
　　(XYZ召唤)
　　XYZ怪兽的正规召唤方法。把场上相同等级的数只怪兽（不可以使用衍生物，可以使用陷阱怪兽）作为素材，从额外卡组特殊召唤。
　　(XYZ素材)
　　XYZ召唤的素材怪兽不送去墓地，而是在XYZ召唤的怪兽卡的下面重叠。功能类似指示物，不当作怪兽使用。XYZ素材被取除时是送去墓地。
详细调整介绍
　　1 XYZ怪兽的素材怪兽在场上集齐
　　2 素材怪兽在1个怪兽卡区域上纵向重叠
　　3 XYZ怪兽用攻击表示或守备表示在那组素材上面重叠
XYZ怪兽的XYZ召唤
　　◇XYZ召唤在同1回合没有次数限制
　　◇XYZ召唤在自己场上的怪兽卡区域没有空位也能进行
　　◇XYZ召唤时，素材怪兽的表示形式不用一致，攻击表示和守备表示的怪兽都可以用
　　◇XYZ召唤时不能用里侧表示怪兽作为XYZ素材
　　◇XYZ召唤时不能用衍生物作为XYZ素材
　　◇XYZ召唤时可以用陷阱怪兽作为XYZ素材
　　◇XYZ召唤时可以用『从场上离开的场合从游戏中除外』等的怪兽作为XYZ素材，这时怪兽不会从游戏中除外而是在XYZ召唤的怪兽的卡下重叠 　　例
　　「螺丝刺猬」「僵尸带菌者」（从场上离开的场合从游戏中除外）
XYZ怪兽的XYZ素材
　　◇XYZ素材在怪兽卡区域上的XYZ怪兽下面纵向重叠（公开情报，双方可以确认）
　　◇重叠不是卡的效果，和「技能吸收」等无关，XYZ素材不会因为XYZ怪兽的效果被无效而消失
　　◇重叠的XYZ素材不当作卡（怪兽），和针对卡（怪兽）的卡无关
　　例1
　　对方场上的卡只有「No.39 希望皇 霍普」（卡下有「召唤僧」「杀人番茄」2个XYZ素材）1张
　　自己的「邪神 抹灭者」（攻击力・守备力变成对方场上存在的卡的数量×1000）的攻击力・守备力是1000
　　自己的「自业自得」（对方场上存在的怪兽数量×500的伤害）的伤害是500
　　例2
　　「雷破」可以选择XYZ怪兽，但不能选择XYZ怪兽下面的卡
　　「爆导索」的效果影响到XYZ怪兽，并不会也影响到XYZ怪兽下面的卡
　　例3
　　「嵌合要塞龙」不能把重叠的「电子龙」和机械族怪兽送去墓地特殊召唤
　　◇XYZ素材被效果等从场上『取除』的场合，那个XYZ素材是送去墓地
　　◇XYZ召唤的怪兽不在场上存在的场合，重叠的XYZ素材从场上送去墓地
　　例
　　XYZ召唤的怪兽被破坏的场合
　　XYZ召唤的怪兽被「亚空间物质传送装置」暂时除外的场合
　　XYZ召唤的怪兽因「强制脱出装置」的回到手卡效果而回到额外卡组的场合
　　◇XYZ召唤的怪兽在场上里侧表示存在的场合，重叠的XYZ素材继续以表侧放在下面
　　◇XYZ召唤的怪兽因控制权转移而去另外的怪兽卡区域的场合，重叠的XYZ素材也一起转移去那个怪兽卡区域上
　　◇「螺丝刺猬」「僵尸带菌者」等XYZ素材从场上离开的场合是送去墓地
　　◇「大宇宙」（送去墓地的卡不去墓地从游戏中除外）等的效果适用中XYZ素材从场上离开的场合，XYZ素材从游戏中除外
　　◇「次元的裂缝」（送去墓地的怪兽不去墓地从游戏中除外）的效果适用中XYZ素材从场上离开的场合，XYZ素材送去墓地
　　◇重叠状态的XYZ素材不当作卡，从场上离开时不是作为卡从场上离开，不符合『卡（怪兽）从场上～～』这类条件的卡
　　例
　　「三眼怪」条件是『这张卡从场上送去墓地时』，但作为「No.17 修正龙」下面的XYZ素材被取除时，「三眼怪」不算『这张卡从场上送去墓地』而不发动效果
　　◇重叠状态的XYZ素材从场上离开时是作为卡送去墓地，符合『卡（怪兽）送去墓地』这类条件的卡
　　例
　　「蒲公英狮」条件是『这张卡送去墓地时』，作为「No.17 修正龙」下面的XYZ素材被取除时不算『这张卡从场上送去墓地』，但算『这张卡送去墓地』，会发动效果
XYZ怪兽没有等级
　　◇XYZ怪兽不能作为同调素材等以等级为条件的召唤素材
　　◇XYZ怪兽不能成为针对等级的效果的对象
　　例
　　XYZ怪兽不能装备「同调英雄」（等级上升1星）和「降格处分」（等级下降2星）等
　　◇XYZ怪兽不受针对等级的效果影响
　　例
　　「等级限制B地区」（4星以上的怪兽变成守备表示）和「超重力网」（4星以上的怪兽不能攻击）等的效果对4阶的「No.39 希望皇 霍普」没有影响
